## Overview

This repository contains a documentation sample for refactoring REST APIs in **ReadyAPI**.

The project recreates a product-style documentation page in Markdown and is structured for publishing with GitHub Pages. It demonstrates task-based writing, clear procedural guidance, screenshot integration, and docs-as-code workflow practices.
